# Smart Parking Management System

## Introduction
This project implements a Smart Parking Management System using computer vision techniques. The system monitors a parking lot via a video feed, identifies occupied and vacant parking spaces, and displays the number of free spaces in real-time. This can help drivers quickly find available spots and optimize parking lot utilization.

## Features
- Real-time detection of occupied and vacant parking spaces.
- Visual annotation of parking spaces on the video feed.
- Displays the total number of free parking spaces.
- Customizable parking space regions using a dedicated picker tool.

## Setup
To set up and run this project, follow these steps:

### Prerequisites
- Python 3.x
- OpenCV
- NumPy
- Ultralytics (for potential future enhancements, though not directly used in the current basic detection logic)

### Installation
1.  Clone the repository:
    ```bash
    git clone <repository_url>
    cd smart-parking-management
    ```
2.  Install the required Python packages:
    ```bash
    pip install -r requirements.txt
    ```

### Project Files
- `main.py`: The main script for running the parking space detection.
- `utils.py`: Contains utility functions, specifically `check_parking_space` for processing parking areas.
- `ParkingSpacePicker.py`: A utility script to define and save the coordinates of parking spaces.
- `requirements.txt`: Lists all necessary Python dependencies.
- `carParkImg.png`: An image of the parking lot (required for `ParkingSpacePicker.py`).
- `parking_video.mp4`: The video feed of the parking lot (required for `main.py`).
- `CarParkPos`: A binary file generated by `ParkingSpacePicker.py` storing the coordinates of defined parking spaces.

## Usage

### 1. Define Parking Spaces (First-time setup or modification)
Before running the main detection, you need to define the parking space regions. This is done using the `ParkingSpacePicker.py` script.

1.  Place an image of your parking lot named `carParkImg.png` in the project directory.
2.  Run the picker script:
    ```bash
    python ParkingSpacePicker.py
    ```
3.  A window will appear showing `carParkImg.png`. Click the left mouse button to mark the top-left corner of a parking space. Click the right mouse button to remove a previously marked space. The coordinates will be saved automatically to `CarParkPos`.
4.  Press `q` to quit the picker.

### 2. Run the Smart Parking System
Once the parking spaces are defined, you can run the main detection system.

1.  Ensure you have a video file named `parking_video.mp4` in the project directory.
2.  Run the main script:
    ```bash
    python main.py
    ```
3.  The system will process the video, detect occupied/vacant spaces, and display the results. (Note: In the current simulation, `cv2.imshow` is commented out, so you won't see a live window, but the processing occurs.)

## How it Works (Brief Technical Overview)

The system works by:
1.  **Loading Parking Space Coordinates**: The `ParkingSpacePicker.py` script allows users to manually select rectangular regions corresponding to parking spots. These coordinates are saved.
2.  **Video Processing**: The `main.py` script reads frames from a video feed.
3.  **Image Preprocessing**: Each frame is converted to grayscale, blurred, and then thresholded to highlight differences between occupied and vacant spots. Morphological operations (dilation) are applied to refine the image.
4.  **Space Status Check**: The `check_parking_space` function in `utils.py` iterates through the defined parking regions. For each region, it counts the number of non-zero pixels (representing car presence after thresholding). A threshold value (e.g., `count < 900`) determines if a space is free or occupied.
5.  **Annotation**: Rectangles are drawn around each parking space, colored green for free and red for occupied. The count of non-zero pixels and the total number of free spaces are displayed.
